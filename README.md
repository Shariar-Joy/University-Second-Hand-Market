# University-Second-Hand-Marketplace
 
Industry-grade and university-ready software engineering documentation repository for the **University Second-Hand Marketplace**.
 
## Repository Purpose
This repository demonstrates a complete **Software Development Life Cycle (SDLC)** and Requirements Engineering journey:
 
Problem Discovery -> Requirement Elicitation -> Product Requirements Document (PRD) -> User Stories -> Requirements -> Software Requirements Specification (SRS) -> Design -> Technical Design Document (TDD) -> Database -> Application Programming Interface (API) -> Implementation -> Testing -> Release
 
## Technology Stack
| Layer | Technology |
|---|---|
| Frontend | React |
| Backend | Python FastAPI |
| Database | PostgreSQL (SQL) |
| Authentication | JSON Web Token (JWT) |
| Image Storage | Cloudinary (or local storage) |
| Deployment | Docker |
