# Second-Hand Marketplace - Survey Report

## Survey Overview
| Attribute | Value |
|---|---|
| Sample Size | ~50 respondents |
| Segments | Students (55%), Young Professionals (30%), Casual Sellers (15%) |
| Duration | 7 days |
| Method | Online structured questionnaire |

## Survey Questions and Results
| Q# | Question | Top Responses | Percentage |
|---|---|---|---|
| Q1 | How often do you buy/sell second-hand items? | Monthly or more | 64% |
| Q2 | Where do you currently trade second-hand goods? | Facebook groups / WhatsApp | 78% |
| Q3 | Biggest problem with current channels? | Cannot search or filter | 71% |
| Q4 | Most desired feature? | Search and category filter | 83% |
| Q5 | Do you need to message sellers privately? | Yes | 89% |
| Q6 | Do you want to save listings for later? | Yes | 76% |
| Q7 | Would a seller dashboard help you manage listings? | Yes | 81% |
| Q8 | Acceptable login method? | Email/password | 92% |
| Q9 | How important is mobile-responsive web? | Very important | 85% |
| Q10 | Would you use a platform with photos + price + condition? | Yes | 94% |

## Feature Demand Ranking
| Rank | Feature | Demand Score (/100) |
|---|---|---|
| 1 | Search by keyword | 91 |
| 2 | Category filter | 88 |
| 3 | Private messaging | 87 |
| 4 | Photo upload for listings | 85 |
| 5 | Seller dashboard | 81 |
| 6 | Favorites / saved listings | 76 |

## Analysis
1. Search and filtering are the strongest adoption drivers for buyers.
2. Private in-platform messaging is critical for seller and buyer safety.
3. Photo upload and condition field are the minimum trust signals for buyers.
4. Seller dashboard is valued by anyone who lists more than 1–2 items.

## Recommendations
| Recommendation | Linked Requirement IDs |
|---|---|
| Build search and filter first in browse flow | FR-022..FR-026 |
| Implement messaging in MVP | FR-029..FR-033 |
| Require photo and condition on listing form | FR-009, FR-016, FR-017 |
| Include seller dashboard with stats | FR-034..FR-036 |
